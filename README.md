# Calculator App

This is a simple Java console-based calculator application.

The user enters two numbers, selects an operation, and the result is displayed.  
The program runs in a loop until the user chooses to exit.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Loop-based execution
- Exit option (yes / no)
- Division by zero check

## How to Run

Compile the program:
```bash
javac Calculator.java
Run the program:

java calculator.Calculator
How to Use
Enter the first number

Enter the second number

Choose an operation:

1 → Addition

2 → Subtraction

3 → Multiplication

4 → Division

View the result

Type yes to exit or no to continue
